# Rocky v1.3.0

Rocky 1.3.0 is a release-hardening upgrade rather than a new-agent-loop rewrite.

## What changed

- Restored tracked `/agent-testing` scenario specs so the repo's live-learning harness is complete again.
- Added a machine-readable capability inventory plus a human-readable scenario catalog.
- Centralized version metadata and made `import rocky` lighter for version-only consumers.
- Added missing release assets: license, changelog, contributor guide, manifest, editor config, and CI workflow.
- Documented a 100-item upgrade audit in `docs/upgrade/rocky-v1.3.0-audit.md`.

## Why the bump is minor

The package surface gained new tracked assets, new release-maintenance scripts, and a new public `rocky.capabilities` module. Those are additive changes, so 1.3.0 is a better fit than a patch bump.
