# xlfg Knowledge Base — rocky

This directory holds durable knowledge for xlfg-driven development runs.

## Structure

- `knowledge/` — tracked durable knowledge (current state, decisions, patterns, etc.)
- `knowledge/agent-memory/` — role-specific memory files
- `runs/` — local-only run artifacts (gitignored)
- `migrations/` — schema or scaffold migration notes
- `meta.json` — scaffold version and metadata
