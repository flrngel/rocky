# UX Flows

## REPL input
- Enter: submit input
- Alt+Enter (Escape+Enter): insert newline (multiline input)
- Ctrl+C / Ctrl+D: exit
- Tab after `/`: slash command completion
- Up/Down: history navigation

## Permission prompt
- Appears as `Allow {family}:{action}? [y/N]`
- Enter submits directly (multiline=False override)
- Accepts "y" or "yes" (case-insensitive)
